## 🎵 Carta Index Interactiva con Música

Este es un proyecto interactivo diseñado con **HTML5** y **CSS3** para crear una experiencia visual y auditiva única. 

### 🌟 Características:
* **Diseño Adaptable:** Estructura limpia y diseño estilizado con CSS.
* **Interactividad:** Animaciones y componentes interactivos para el usuario.
* **Ambiente Musical:** Incorporación de audio de fondo (Oasis 🎶) mediante controles web optimizados.
